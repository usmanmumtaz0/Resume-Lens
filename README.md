# 🔍 ResumeLens - AI Resume Analyzer

**ResumeLens** is a professional AI-powered resume information extractor built with **Streamlit** and **Google Gemini API**. It quickly extracts essential information from resumes and displays it in an interactive, modern dashboard.

---

## 🚀 Features

- Extract **Name**, **Address**, **Contact**, and **Projects** from resumes.
- Modern **SaaS-style dashboard** with cards for each section.
- **Light & Dark mode** with professional colors for readability.
- **Download results** in JSON or text format.
- Interactive sidebar for easy navigation.
- Clean and professional UI designed for recruiters and developers.

---

## 💻 Tech Stack

- **Frontend & App:** Streamlit  
- **AI & NLP:** LangChain, Google Gemini API  
- **Environment:** Python 3.9+  
- **Security:** `.env` for API keys, `.gitignore` to avoid uploading sensitive files
